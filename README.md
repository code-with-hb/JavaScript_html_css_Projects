# JAVASCRIPT, CSS, HTML PROJECTS

A collection of modern and reusable UI components built with **HTML5**, **CSS3**, and **JavaScript**. This repository showcases creative user interface elements that can be integrated into websites to improve aesthetics, usability, and user experience.

---

## 📖 Overview

This project contains handcrafted UI components designed to demonstrate modern front-end development techniques. Each component is built from scratch using vanilla HTML, CSS, and JavaScript without relying on external frameworks.

Whether you're learning front-end development or looking for reusable UI ideas, this repository serves as a practical reference.

---

## ✨ Included Projects

### 🎯 Custom Scrollbar

A fully customized scrollbar designed to replace the browser's default scrollbar with a cleaner and more modern appearance.

**Features**

* Modern scrollbar design
* Smooth scrolling experience
* Lightweight implementation
* Responsive layout
* Easy to customize
* Cross-browser friendly (where supported)

---

### 🚀 Creative Menus

A collection of stylish navigation menus featuring modern layouts, hover animations, and smooth transitions.

**Features**

* Multiple creative navigation styles
* Smooth hover animations
* Responsive design
* Modern UI principles
* Clean and reusable code
* Easy customization
